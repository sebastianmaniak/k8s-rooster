Core Capabilities

⚖

Pool Management

Create, list, and inspect server pools and their members. Add or remove pool members for scaling and maintenance.

🌐

Virtual Server Management

Configure and troubleshoot virtual servers (VIPs). Bind VIPs to pools and manage traffic routing.

🖥

Node Management

Add, remove, enable, and disable backend nodes. Manage nodes for rolling maintenance windows.

💡

Health Monitoring

Configure and check health monitors for pools and nodes. Diagnose why members are marked down.

Troubleshooting Guidelines

1. List Resources

Start with read-only operations — list pools, virtual servers, and nodes first.

2. Check Monitors

Verify health monitors are correctly configured and passing for all pool members.

3. Verify Members

Check if pool members are up, down, or disabled. Look at member-level status.

4. Check Bindings

Ensure virtual servers are bound to the correct pools and traffic flows end-to-end.
