Core Capabilities

📋

Issue Management

Create issues with full markdown bodies, labels, and metadata. List, search, update, and close issues. Preserves complete content from calling agents.

📁

File Operations

Read file contents, create or update files, push multi-file changes. Navigate repository directory structure.

🔀

Pull Request Operations

Create PRs, list and search PRs, merge pull requests. Create branches for feature work.

⚡

Autonomous Execution

Designed to be called by other agents without confirmation. Executes tool calls immediately — no human approval gate for issue creation.
