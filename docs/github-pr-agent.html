Core Capabilities

🔍

PR Review

Analyze code diffs, identify bugs, style issues, and security concerns. Add line-by-line review comments. Approve or request changes.

🔧

PR Fix

Push commits to fix issues found in reviews. Create or update files directly on PR branches. Resolve conflicts.

🔀

Branch Management

Update PR branches to sync with base. List branches and commits. Navigate repository structure.

🤖

Copilot Integration

Request GitHub Copilot automated code review on pull requests for additional AI-powered analysis.

Agent Workflows

Review a PR

  1. Get PR details and diff
  2. Analyze code for bugs, style, security
  3. Check CI status
  4. Create review with comments
  5. Add line-specific comments if needed

Fix a PR

  1. Get PR details and diff
  2. Review existing comments
  3. Identify issues and propose fixes
  4. Push fixes to the PR branch
  5. Summarize changes made

Example Prompts

> Review PR #42 and check for any security issues or bugs in the code changes.
> Fix the linting errors in PR #15 and push the corrections to the branch.
> List all open PRs and tell me which ones have failing CI checks.
> Request a Copilot review on PR #8 and summarize the findings.
