Troubleshooting Framework

1. Check Versions

Verify Istio CLI, control plane, and data plane versions match. Version skew causes subtle issues.

2. Proxy Status

Check proxy injection and sync status. Look for NOT_SENT or STALE configs in proxy status output.

3. Analyze Config

Run istio_analyze_cluster_configuration to catch misconfigurations, missing resources, and policy conflicts.

4. Check mTLS

Verify PeerAuthentication policies. Mixed STRICT/PERMISSIVE modes across namespaces often cause 503s.
