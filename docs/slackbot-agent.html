Core Capabilities

💬

Slack Integration

Posts formatted messages, updates, and reports to Slack channels. Users interact via Slack messages using the A2A protocol.

🔍

Kubernetes Diagnostics

Query resources, describe pods, read logs, check events, verify service connectivity, and execute commands inside pods across all namespaces.

⚖

Helm Release Management

List Helm releases and get detailed release information. Check chart versions, values, and deployment status.

📋

Slack-Optimized Output

Concise, scannable responses formatted for Slack. Code blocks for YAML/logs, bullet points for lists, summaries before details.
